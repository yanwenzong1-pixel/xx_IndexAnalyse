## 🚀 快速开始

### 步骤 1：启动 Web 服务器

**方法 1：使用 Python**
```bash
cd web/frontend
python -m http.server 5500
```

**方法 2：使用一键启动脚本**
```bash
# Windows
启动系统.py

# 或手动启动
cd web/frontend
python -m http.server 5500
```

### 步骤 2：访问系统

浏览器打开：`http://localhost:5500/index.html`

## 💾 数据保存流程

### 保存数据到本地

1. **点击"📝 数据录入"按钮**
2. **填写或粘贴数据**
   - 单条录入：逐个字段填写
   - 批量导入：粘贴 JSON 数组或东方财富 JSONP 格式
3. **点击"💾 保存"按钮**
4. **浏览器自动下载 `index_history_data.json` 文件**
5. **手动覆盖文件**：
   - 找到下载的文件（通常在"下载"文件夹）
   - 复制到项目目录：`web/frontend/index_history_data.json`
   - 覆盖原文件
6. **刷新浏览器页面**（F5 或 Ctrl+R）

## 🔄 数据更新流程

### 方式 1：使用数据录入功能

1. 打开系统页面
2. 点击"📝 数据录入"
3. 粘贴东方财富数据（JSONP 格式或 JSON 数组）
4. 点击"💾 保存"
5. 浏览器下载 `index_history_data.json`
6. 手动覆盖到 `web/frontend/` 目录
7. 刷新页面

### 方式 2：直接编辑文件

1. 打开 `web/frontend/index_history_data.json`
2. 使用文本编辑器修改数据
3. 保存文件
4. 刷新浏览器页面

## 🛠️ 常见问题

### Q1: 保存时没有下载文件？

**检查：**
1. 浏览器是否阻止了下载
2. 查看浏览器地址栏右侧是否有下载图标
3. 检查浏览器的"下载内容"（Ctrl+J）

### Q2: 覆盖了文件但页面没更新？

**解决方案：**
1. 强制刷新浏览器（Ctrl + F5）
2. 清除浏览器缓存
3. 检查文件路径是否正确

### Q3: 如何验证数据已生效？

**方法：**
1. 打开 `web/frontend/index_history_data.json`
2. 查看最后几条数据是否是刚保存的
3. 刷新页面，检查最新数据

### Q4: 可以删除后端目录吗？

**可以！** 后端服务已完全移除，可以安全删除：
```
web/backend/          # 可删除
启动后端服务.bat      # 可删除
test_save_api.py      # 可删除
```

```batch
copy "%USERPROFILE%\Downloads\index_history_data.json" "web\frontend\index_history_data.json" /Y
echo 数据已更新！
pause
```
