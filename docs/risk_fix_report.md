## 🔧 修复内容详情

### 修复项 1：单日暴跌风险计算 ⭐⭐⭐

**问题**：原公式完全不考虑单日涨跌幅风险

**修复**：
```python
# 新增单日暴跌风险计算
daily_change = df['change_pct'].iloc[-1]

if daily_change < -7:
    risk_score += 2.0
elif daily_change < -5:
    risk_score += 1.5  # -5.98% 触发此档
elif daily_change < -3:
    risk_score += 1.0
elif daily_change < -2:
    risk_score += 0.5
```

**效果**：-5.98% 暴跌额外 +1.5 分

---

### 修复项 2：价格趋势风险双向计算 ⭐⭐⭐

**问题**：原公式只计算上涨风险，忽略下跌风险

**修复前**：
```python
if recent_change > 30:  # 只计算上涨
    risk_score += 2.0
```

**修复后**：
```python
# 上涨风险
if recent_change > 15:
    risk_score += min(2.0, 1.0 + (recent_change - 15) / 10.0)

# 下跌风险（新增）
if recent_change < -15:
    risk_score += min(2.0, abs(recent_change) / 10.0)
elif recent_change < -8:
    risk_score += abs(recent_change) / 10.0
```

**效果**：5 日累计下跌 -12.28% 额外 +1.2 分

---

### 修复项 3：连续下跌风险计算 ⭐⭐

**问题**：原公式未考虑连续下跌累积风险

**修复**：
```python
# 新增连续下跌风险计算
consecutive_down = 0
for i in range(min(5, len(df))):
    if df['change_pct'].iloc[-(i+1)] < 0:
        consecutive_down += 1
    else:
        break

if consecutive_down >= 5:
    risk_score += 2.0
elif consecutive_down >= 3:
    risk_score += 1.0
```

**效果**：连续 5 日下跌额外 +2.0 分

---

### 修复项 4：优化流动性风险阈值 ⭐

**修复前**：
```python
if avg_turnover_5d > 15:  # 阈值过高
    risk_score += 2.0
```

**修复后**：
```python
if avg_turnover_5d > 12:  # 降低阈值
    risk_score += 2.0
elif avg_turnover_5d > 8:
    risk_score += 1.0 + (avg_turnover_5d - 8) / 4.0
elif avg_turnover_5d > 5:  # 新增 5% 档位
    risk_score += (avg_turnover_5d - 5) / 3.0
```

**效果**：换手率风险敏感度提升

---

### 修复项 5：均线跌破风险阈值优化 ⭐

**修复前**：
```python
if drop_ratio > 0.1:  # 跌破 10% 才加 2 分
    risk_score += 2.0
```

**修复后**：
```python
if drop_ratio > 0.05:  # 跌破 5% 加 2 分
    risk_score += 2.0
elif drop_ratio > 0.02:  # 跌破 2% 开始加分
    risk_score += 1.0 + (drop_ratio - 0.02) / 0.03
```

**效果**：均线跌破敏感度提升 2.5 倍

---

### 修复项 6：波动率风险阈值优化 ⭐
