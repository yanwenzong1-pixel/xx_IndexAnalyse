# 系统架构文档

## 1. 整体架构

### 1.1 架构分层

```
┌─────────────────────────────────────┐
│         前端层 (Frontend)            │
│  ┌─────────────────────────────┐    │
│  │  Pages (页面层)              │    │
│  │  Components (组件层)         │    │
│  └─────────────────────────────┘    │
└─────────────────────────────────────┘
                 ↕ HTTP/JSON
┌─────────────────────────────────────┐
│         后端层 (Backend)             │
│  ┌─────────────────────────────┐    │
│  │  API (接口层)                │    │
│  │  Service (业务逻辑层)        │    │
│  │  Model (数据模型层)          │    │
│  │  DAO (数据访问层)            │    │
│  │  Core (核心配置层)           │    │
│  └─────────────────────────────┘    │
└─────────────────────────────────────┘
                 ↕
┌─────────────────────────────────────┐
│         数据层 (Data)                │
│  - 东方财富 API                      │
│  - 本地缓存 (JSON 文件)              │
│  - LocalStorage (前端)              │
└─────────────────────────────────────┘
```

## 2. 后端架构

### 2.1 分层设计

#### API 层 (api/)
- 职责：接收请求、参数验证、响应格式化
- 文件：`data_api.py`, `indicator_api.py`, `risk_api.py`, `report_api.py`
- 规范：每个 API 文件对应一个业务模块

#### Service 层 (service/)
- 职责：业务逻辑处理、数据计算
- 文件：
  - `data_fetch_service.py` - 数据获取
  - `indicator_service.py` - 指标计算
  - `risk_service.py` - 风险评估
  - `decline_prediction_service.py` - 下跌预测
  - `report_service.py` - 报告生成

#### Model 层 (model/)
- 职责：数据模型定义、响应格式
- 文件：
  - `stock_model.py` - 股票数据模型
  - `response_model.py` - API 响应模型

#### DAO 层 (dao/)
- 职责：数据访问、缓存管理
- 文件：
  - `data_dao.py` - 数据获取
  - `cache_dao.py` - 缓存操作

#### Core 层 (core/)
- 职责：配置管理、常量定义
- 文件：
  - `config.py` - 配置
  - `api_config.py` - API 配置

## 4. 数据流

### 4.1 数据获取流程

```
用户请求
  ↓
API 层 (接收请求)
  ↓
Service 层 (业务逻辑)
  ↓
DAO 层 (数据访问)
  ↓
缓存检查 → 有缓存 → 返回缓存
  ↓ 无缓存
外部 API (东方财富)
  ↓
数据解析
  ↓
缓存写入
  ↓
返回响应
```

### 4.2 前端数据流

```
组件发起请求
  ↓
api/index.js (封装)
  ↓
后端 API
  ↓
响应处理
  ↓
状态更新
  ↓
缓存保存 (localStorage)
  ↓
UI 渲染
```

## 5. 缓存策略

### 5.1 后端缓存

| 数据类型 | TTL | 存储位置 |
|---------|-----|---------|
| 股票数据 | 30 分钟 | JSON 文件 |
| 指标数据 | 5 分钟 | JSON 文件 |
| 风险数据 | 30 分钟 | JSON 文件 |

### 5.2 前端缓存

| 数据类型 | TTL | 存储位置 |
|---------|-----|---------|
| 股票数据 | 3 小时 | localStorage |
| 风险数据 | 3 小时 | localStorage |
| Tab 状态 | 7 天 | localStorage |
