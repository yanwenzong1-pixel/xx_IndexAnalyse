# 风险打分异常问题分析报告

## 📋 问题概述

**现象**：市场连续暴跌（-5.98%），但风险打分停留在 6.8 分，未能及时反映市场风险

**预期**：暴跌情况下风险打分应显著提升（至少 8 分以上）

**影响**：风险评估失真，无法为用户提供准确的风险预警

---

## 🔍 根本原因分析

### 1. 计算公式缺陷

#### 1.1 基础分设置不合理

**当前逻辑**：
```python
risk_score = 5.0  # 基础风险分数
```

**问题**：
- 基础分 5.0 属于"中等风险"区间
- 即使所有风险指标都正常，起步就是 5 分
- 导致风险打分"易升难降"，敏感度不足

**建议**：
- 基础分应设为 3.0（低风险区间）
- 让风险分数根据实际指标动态上升

---

#### 1.2 流动性风险阈值过高

**当前逻辑**：
```python
avg_turnover_5d = df['turnover'].tail(5).mean()

if avg_turnover_5d > 15:
    risk_score += 2.0
elif avg_turnover_5d > 8:
    risk_score += 1.0 + (avg_turnover_5d - 8) / 7.0
elif avg_turnover_5d > 3:
    risk_score += (avg_turnover_5d - 3) / 5.0
```

**问题分析**：

| 换手率 | 当前加分 | 问题 |
|--------|---------|------|
| 3% 以下 | 0 分 | ✅ 合理 |
| 3-8% | 0-1 分 | ⚠️ 阈值偏高 |
| 8-15% | 1-2 分 | ⚠️ 阈值偏高 |
| 15% 以上 | 2 分 | ❌ 封顶过早 |

**实际情况**：
- 微盘股正常换手率：3-5%
- 高风险换手率：8-12%
- 极端换手率：15%+

**当前截图数据**：换手率 3.42%（较低风险）
- 按当前公式：加分 ≈ 0.08 分
- **实际应加分**：应考虑换手率骤降的流动性风险

---

#### 1.3 价格趋势风险计算错误 ❌❌

**当前逻辑**：
```python
recent_change = df['change_pct'].tail(5).sum()

if recent_change > 30:
    risk_score += 2.0
elif recent_change > 20:
    risk_score += 1.0 + (recent_change - 20) / 10.0
elif recent_change > 10:
    risk_score += (recent_change - 10) / 10.0
```

**严重问题**：
1. **只计算上涨风险，未计算下跌风险！**
2. 阈值设置：10%、20%、30% 都是针对"上涨"
3. 暴跌情况下（如 -5.98%），此逻辑不加分！

**实际数据验证**：
- 假设近 5 日涨跌幅：[-5.98%, -3%, -2%, -1%, 0%]
- 累计：-11.98%
- 当前公式：**不加分**（因为<10%）
- **应有风险**：暴跌 12% 应加 2 分！

**修复方案**：
```python
recent_change = df['change_pct'].tail(5).sum()

# 上涨风险
if recent_change > 10:
    risk_score += min(2.0, recent_change / 10.0)

# 下跌风险（新增）
if recent_change < -10:
    risk_score += min(2.0, abs(recent_change) / 10.0)
elif recent_change < -5:
    risk_score += abs(recent_change) / 10.0
```

---

#### 1.4 均线跌破风险阈值不合理

**当前逻辑**：
```python
if close_price < ma20_price:
    drop_ratio = (ma20_price - close_price) / ma20_price
    if drop_ratio > 0.1:
        risk_score += 2.0
    elif drop_ratio > 0.05:
        risk_score += 1.0 + (drop_ratio - 0.05) / 0.05
    else:
        risk_score += drop_ratio / 0.05
```

**问题**：
- 阈值设置过高（5%、10%）
- 微盘股波动大，跌破 2% 就应警惕
- 当前市场可能只跌破 1-2%，加分微乎其微

**建议调整**：
```python
if drop_ratio > 0.05:
    risk_score += 2.0
elif drop_ratio > 0.02:
    risk_score += 1.0 + (drop_ratio - 0.02) / 0.03
else:
    risk_score += drop_ratio / 0.02
```

---

#### 1.5 波动率风险阈值过高

**当前逻辑**：
```python
volatility = df['change_pct'].rolling(20).std().iloc[-1]

if volatility > 10:
    risk_score += 2.0
elif volatility > 5:
    risk_score += 1.0 + (volatility - 5) / 5.0
elif volatility > 3:
    risk_score += (volatility - 3) / 2.0
```

**问题**：
- 微盘股正常波动率：3-5%
- 高风险波动率：5-8%
- 极端波动率：8%+

**当前阈值**：3%、5%、10%
- 10% 封顶过早
- 应降低阈值为 5%、8%

---

### 2. 缺失关键风险因子

#### 2.1 单日暴跌风险 ❌

**问题**：当前公式**完全没有考虑单日涨跌幅风险**！

**实际情况**：
- 截图显示：-5.98%（暴跌）
- 当前公式：不加分
- **应有加分**：至少 +1.5 分

**建议新增**：
```python
daily_change = df['change_pct'].iloc[-1]

if daily_change < -5:
    risk_score += 2.0
elif daily_change < -3:
    risk_score += 1.5
elif daily_change < -2:
    risk_score += 1.0
```

---

#### 2.2 成交量异常风险 ❌

**问题**：未考虑成交量骤变风险

**实际情况**：
- 暴跌时成交量放大 = 恐慌性出逃
- 成交量萎缩 = 流动性枯竭
- 都应提升风险等级

**建议新增**：
```python
avg_amount_20d = df['amount'].rolling(20).mean().iloc[-1]
current_amount = df['amount'].iloc[-1]
amount_ratio = current_amount / avg_amount_20d

# 成交量异常放大（>150%）
if amount_ratio > 1.5:
    risk_score += 1.0

# 成交量异常萎缩（<50%）
elif amount_ratio < 0.5:
    risk_score += 1.5
```

---

#### 2.3 连续下跌风险 ❌

**问题**：未考虑连续下跌天数

**实际情况**：
- 连续 3 日下跌：风险累积
- 连续 5 日下跌：高风险

**建议新增**：
```python
consecutive_down = 0
for i in range(5):
    if df['change_pct'].iloc[-(i+1)] < 0:
        consecutive_down += 1
    else:
        break

if consecutive_down >= 5:
    risk_score += 2.0
elif consecutive_down >= 3:
    risk_score += 1.0
```

---

### 3. 风险权重分配不合理

**当前权重**（估算）：

| 风险维度 | 当前权重 | 问题 |
|---------|---------|------|
| 流动性 | 20% | ⚠️ 偏低 |
| 价格趋势 | 30% | ⚠️ 只考虑上涨 |
| 波动率 | 20% | ⚠️ 阈值过高 |
| 单日暴跌 | 0% | ❌ 缺失 |
| 成交量 | 0% | ❌ 缺失 |
| 连续下跌 | 0% | ❌ 缺失 |

**建议权重**：

| 风险维度 | 建议权重 | 调整 |
|---------|---------|------|
| 流动性 | 20% | - |
| 价格趋势 | 25% | ↓ |
| 波动率 | 15% | ↓ |
| 单日风险 | 20% | ↑ 新增 |
| 成交量 | 10% | ↑ 新增 |
| 连续下跌 | 10% | ↑ 新增 |
