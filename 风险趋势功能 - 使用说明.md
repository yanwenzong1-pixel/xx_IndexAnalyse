## 🔍 问题诊断

### 如果点击"风险趋势"后没有显示图表

#### 方法 1：检查浏览器控制台

1. 按 `F12` 打开开发者工具
2. 切换到 **Console（控制台）** 标签
3. 查看是否有以下日志：

**正常情况**应看到：
```
[RiskTab] 使用缓存数据
```
或
```
[RiskTab] 开始加载风险数据...
[RiskTab] API 响应状态：200
[RiskTab] API 返回结果：{success: true, data: Array(252), ...}
[RiskTab] 数据加载成功，共 252 条记录
```

**异常情况**：
- 看到红色错误 → 表示请求失败
- 看到 `ERR_CONNECTION_REFUSED` → 后端未启动
- 看到 `CORS policy` 错误 → 跨域问题

#### 方法 2：检查后端服务

1. 确认步骤 1 的终端窗口仍在运行
2. 看到 `正在监听 http://0.0.0.0:5000` 或类似输出
3. 没有错误信息

**如果后端未启动**，重新执行步骤 1。

#### 方法 3：直接测试 API

在浏览器中访问：

```
http://localhost:5000/api/risk/history?days=5
```

**正常情况**应看到 JSON 数据：
```json
{
  "success": true,
  "data": [
    {"date": "2024-01-01", "risk_score": 6.5, "risk_level": "high"},
    ...
  ],
  "metadata": {
    "total_days": 5,
    "avg_risk": 5.2,
    ...
  }
}
```

**异常情况**：
- 显示 `无法访问此网站` → 后端未启动
- 显示错误信息 → 查看错误详情

#### 方法 4：使用测试页面

访问测试页面：

```
http://localhost:3000/test-risk-data.html
```

点击"加载风险数据"按钮，查看测试结果。

## 📊 功能特性

### 1. 数据展示
- **X 轴**：日期（最近 252 个交易日）
- **Y 轴**：风险打分（0-10 分）
- **折线**：红色实线，平滑曲线
- **Tooltip**：鼠标悬停显示详细信息

### 2. 风险等级
| 分数 | 等级 | 含义 |
|------|------|------|
| 1-3 | 低风险 | 市场风险较低 |
| 4-5 | 中等风险 | 市场风险适中 |
| 6-7 | 较高风险 | 市场风险较高 |
| 8-10 | 高风险 | 市场风险很高 |

### 3. 缓存机制
- **首次加载**：从 API 获取数据（约 1-2 秒）
- **后续访问**：从 localStorage 读取（瞬间）
- **缓存有效期**：24 小时
- **自动刷新**：缓存过期后自动重新获取

### 4. 数据一致性
- 与"综合风险评估"模块使用相同的风险计算逻辑
- 避免前视偏差（每个时点只用历史数据）
- 确保数据准确性

## 🛠️ 常见问题

### Q1: 页面刷新后数据丢失？
**A**: 这是正常的。页面刷新后会重新从 API 加载数据。如果您想使用缓存，请在 24 小时内再次访问。

### Q2: 数据加载很慢？
**A**: 首次加载需要计算 252 天的历史风险，大约需要 1-2 秒。后续访问会使用缓存，瞬间加载。

### Q3: 图表不显示？
**A**: 请按以下顺序检查：
1. 后端服务是否启动
2. 浏览器 Console 是否有错误
3. Network 请求是否成功（状态码 200）
4. 数据格式是否正确

### Q4: 如何清除缓存？
**A**: 在浏览器控制台执行：
```javascript
localStorage.removeItem('riskHistoryData');
localStorage.removeItem('riskHistoryTimestamp');
location.reload();
```

### Q5: 风险分数为什么都一样？
**A**: 如果所有风险分数都相同，可能是：
1. 后端计算逻辑问题 → 检查后端日志
2. 数据源问题 → 检查基础数据

## 📝 技术细节

### API 接口

#### GET /api/risk/history
获取历史风险打分数据

**参数**：
- `days` (int, 可选): 回溯天数，默认 252，范围 1-252

**响应**：
```json
{
  "success": true,
  "data": [
    {
      "date": "2024-01-01",
      "risk_score": 6.5,
      "risk_level": "high"
    }
  ],
  "metadata": {
    "total_days": 252,
    "avg_risk": 5.2,
    "max_risk": 8.3,
    "min_risk": 2.1,
    "current_risk": 6.0,
    "risk_trend": "stable"
  }
}
```

#### GET /api/risk/current
获取当前风险打分（快速接口）

**响应**：
```json
{
  "success": true,
  "data": {
    "date": "2024-01-01",
    "risk_score": 6.5,
    "risk_level": "high"
  }
}
```
