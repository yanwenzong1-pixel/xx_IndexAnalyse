# 微盘股指数分析系统

## 系统功能

本系统是一个基于A股实战投研视角的微盘股指数分析工具，全面、系统、深度、专业地分析当前总市值50亿元以下微盘股指数的涨跌与波动核心驱动指标。

### 核心功能

1. **数据获取与解析**：从东方财富API获取微盘股指数（BK1158）的日K数据
2. **五大维度指标分析**：
   - 流动性维度：日均成交额/总市值、日均换手率、买卖盘价差、流动性覆盖率
   - 资金结构维度：量化成交额占比、融资余额变化
   - 估值与业绩维度：波动率、趋势指标（MA5/MA20）
   - 政策与制度维度：IPO活跃度、监管强度、财报披露窗口期
   - 宏观环境维度：剩余流动性、制造业PMI、市场风险偏好、利率环境
3. **风险评估与预警**：
   - 风险等级评估（1-10级）
   - 下跌风险预测
   - 预警信号触发
4. **可视化界面**：
   - 风险等级显示
   - 指标卡片
   - 价格走势图
   - 换手率和成交额图
   - 预警信息显示
5. **自动报告生成与邮件发送**：
   - 每日收盘后自动生成行情监控报告
   - 发送至指定邮箱

## 技术实现

- **编程语言**：Python 3
- **核心库**：
  - requests：数据获取
  - pandas：数据处理
  - matplotlib：数据可视化
  - seaborn：增强可视化效果
  - tkinter：GUI界面
  - schedule：定时任务
  - smtplib：邮件发送

## 使用方法

1. **安装依赖**：
   ```bash
   pip install requests pandas matplotlib seaborn schedule
   ```

2. **配置邮件发送**：
   - 在`micro_cap_analysis.py`文件中，修改`send_email_report`方法中的邮件配置
   - 设置发送邮箱和密码

3. **运行系统**：
   ```bash
   python micro_cap_analysis.py
   ```

4. **系统操作**：
   - 点击"刷新数据"按钮手动更新数据
   - 系统每3小时自动刷新数据
   - 每日15:30自动生成并发送报告

## 指标说明

### 流动性维度
- **日均成交额/总市值**：衡量市场深度，值越大流动性越好
- **日均换手率**：衡量交易活跃度，值过高可能意味着交易过热
- **买卖盘价差**：衡量市场效率，值越小流动性越好
- **流动性覆盖率**：衡量市场稳定性，值越大流动性风险越小

### 资金结构维度
- **量化成交额占比**：衡量资金参与度
- **融资余额变化**：衡量杠杆资金动向

### 估值与业绩维度
- **波动率**：衡量价格波动程度
- **趋势指标**：MA5和MA20的关系反映短期和中期趋势

### 政策与制度维度
- **IPO活跃度**：影响市场资金分流
- **监管强度**：影响市场情绪
- **财报披露窗口期**：可能带来业绩波动

### 宏观环境维度
- **剩余流动性**：影响市场资金面
- **制造业PMI**：反映经济基本面
- **市场风险偏好**：影响资金流向
- **利率环境**：影响资金成本

## 风险等级划分

- **1-3级**：低风险，绿色显示
- **4-6级**：中等风险，橙色显示
- **7-10级**：高风险，红色显示

## 预警条件

- **下跌预警**：当指数有较大概率下跌＞4%时触发
- **买点信号**：当MA5上穿MA20且当日上涨时触发
