## 🔄 需要重启后端服务

**请按以下步骤操作**：

### 方式 1：手动重启（推荐）

1. **停止当前运行的后端服务**
   - 找到运行 `app.py` 的终端
   - 按 `Ctrl+C` 停止

2. **重启后端服务**
   ```bash
   cd web/backend
   python app.py
   ```

3. **刷新前端页面**
   - 按 `Ctrl+Shift+R` 强制刷新（清除缓存）
   - 或关闭浏览器重新打开

### 方式 2：使用启动脚本

```bash
# 停止旧服务（Ctrl+C）
# 然后运行
python web/backend/app.py
```

## 🔍 故障排查

### 如果风险打分仍然是 6.8 分

**可能原因 1**：后端未重启
- 解决：停止旧服务，重新启动

**可能原因 2**：浏览器缓存
- 解决：按 `Ctrl+Shift+R` 强制刷新
- 或清除浏览器缓存

**可能原因 3**：数据缓存
- 解决：点击"刷新数据"按钮
- 或重启后端服务
