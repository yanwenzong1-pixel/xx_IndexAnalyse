## 六、问题排查

### 6.1 常见问题

**问题 1**: 模块导入失败
```
ModuleNotFoundError: No module named 'requests'
```
**解决**: 安装依赖 `pip install requests pandas numpy scipy`

**问题 2**: 缓存目录不存在
```
FileNotFoundError: [Errno 2] No such file or directory: 'data/cache'
```
**解决**: FactorDataService 会自动创建目录，确保有写权限

**问题 3**: 数据获取失败
```
数据获取失败：...
```
**解决**: 检查网络连接，确认东方财富 API 可访问
