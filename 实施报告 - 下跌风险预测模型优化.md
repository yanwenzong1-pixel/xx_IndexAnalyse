## 三、核心算法实现

### 3.1 因子标准化

**分位数滚动标准化**:
```python
def normalize(self, factor_value, factor_id, window_days=252):
    # 获取历史数据
    history = self.factor_service.get_factor_history(factor_id, window_days)
    
    # 添加当前值到历史数据
    extended_history = history + [factor_value]
    
    # 计算分位数排名
    ranks = rankdata(extended_history, method='average')
    normalized_value = ranks[-1] / len(extended_history)
    
    return min(1.0, max(0.0, normalized_value))
```

---

### 3.2 维度风险分计算

**资金结构维度**:
```python
S_fund = w₁·Norm(X₁) + w₂·Norm(-X₂) + w₃·Norm(X₃) + w₄·Norm(-X₄) + w₅·|Norm(X₅)|
```
其中 X₂和 X₄是负向风险因子（值越大风险越小），使用反向处理

---

### 3.3 趋势调整因子

```python
# 计算各组成部分
ma_ratio = MA5 / MA20  # 短期均线相对中期均线的比值
price_deviation = (Index - MA20) / MA20  # 指数相对中期均线的偏离度
momentum_20d = 20 日动量指标

# 综合趋势因子
trend_factor = 0.4 * ma_ratio + 0.3 * price_deviation + 0.3 * (momentum_20d / 100)

# 使用 Sigmoid 函数将趋势因子映射到 (0,1) 区间
adjustment = 1 / (1 + exp(-k * trend_factor))  # k=2 为敏感系数
```

---

### 3.4 综合风险分计算

```python
# 基础风险分
R_base = ω₁·S_fund + ω₂·S_liquidity + ω₃·S_valuation + ω₄·S_policy + ω₅·S_macro

# 趋势调整后风险分
R_adjusted = R_base · Adjustment
```

---

### 3.5 下跌概率计算

**T+1 模型**:
```python
ln(P/(1-P)) = α₀ + α₁·R_adjusted + α₂·R_adjusted² + α₃·Volatility_20d + α₄·Momentum_1d

P_T+1 = 1 / (1 + exp(-(α₀ + α₁·R_adjusted + α₂·R_adjusted² + α₃·Vol + α₄·Mom)))
```

**参数**:
- α₀ = -1.5 (截距项)
- α₁ = 2.0 (风险分线性项系数)
- α₂ = 0.5 (风险分二次项系数)
- α₃ = 0.03 (波动率系数)
- α₄ = -0.02 (动量系数)

**T+5 模型**:
```python
ln(P/(1-P)) = β₀ + β₁·R_adjusted + β₂·R_adjusted² + β₃·Vol + β₄·Mom_5d + β₅·MacroUncert + β₆·(R_adjusted × Mom_5d)

P_T+5 = 1 / (1 + exp(-(β₀ + β₁·R_adjusted + β₂·R_adjusted² + β₃·Vol + β₄·Mom_5d + β₅·MacroUncert + β₆·(R_adjusted × Mom_5d))))
```

**新增参数**:
- β₅ = 0.05 (宏观不确定性系数)
- β₆ = -0.02 (交互项系数，高风险 + 负动量时概率放大)
