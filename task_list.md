# 任务清单 - 图表容器尺寸修复

## 任务概述

**任务 ID**：TASK-2026-03-10-001  
**任务描述**：修复价格指数图表显示为水平线的 bug，根因是图表容器在初始化时高度为 0 或未完全渲染

### Task 1: 修改 PriceChart.jsx - 容器尺寸检测机制

**Task ID**: TASK-001-01  
**描述**: 在 PriceChart 组件中实现容器尺寸检测机制  
**前置条件**: PRD 和技术实现文档已审批通过  
**输入**: 当前 PriceChart.jsx 文件  
**输出**: 修改后的 PriceChart.jsx 文件  
**验收标准**:
- [ ] 新增 `isContainerReady` 函数，检查容器 offsetWidth 和 offsetHeight
- [ ] 新增 `delayedInit` 函数，实现延迟重试机制（最多 10 次，间隔 100ms）
- [ ] 新增 ResizeObserver 监听容器尺寸变化
- [ ] 修改初始渲染使用 `delayedInit` 而非直接 `renderChart`
- [ ] 修改 forceRefresh 使用 `delayedInit` 而非直接 `renderChart`
- [ ] 代码无 ESLint 警告
