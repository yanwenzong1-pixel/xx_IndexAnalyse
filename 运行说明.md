# 微盘股指数分析系统运行说明

## 系统类型

本系统是一个**桌面应用程序**，不是Web应用，因此**不能**通过浏览器访问 `localhost`。

## 正确运行方式

### 方法一：直接运行
1. 打开命令提示符（CMD）或PowerShell
2. 导航到系统目录：
   ```bash
   cd c:\Users\User\Documents\xx_IndexAnalyse
   ```
3. 运行系统：
   ```bash
   python micro_cap_analysis.py
   ```

### 方法二：使用Python IDE运行
1. 打开Python IDE（如PyCharm、VS Code等）
2. 打开 `micro_cap_analysis.py` 文件
3. 点击运行按钮

## 系统启动后

系统启动后会自动：
1. 获取微盘股指数数据
2. 计算各项指标
3. 显示可视化界面
4. 每3小时自动刷新数据
5. 每日15:30自动生成并发送报告

## 界面说明

- **风险等级**：显示当前风险等级（1-10级），颜色表示风险程度（绿色：低风险，橙色：中等风险，红色：高风险）
- **指标卡片**：显示收盘价、涨跌幅、成交额、换手率等核心指标
- **价格走势图**：显示微盘股指数的价格走势，包含MA5和MA20均线
- **换手率和成交额图**：显示换手率和成交额的变化趋势
- **预警信息**：显示风险警报或买点信号

## 常见问题

### 无法运行
- 检查Python是否已安装：运行 `python --version`
- 检查依赖是否已安装：运行 `pip install requests pandas matplotlib seaborn schedule`
- 检查网络连接是否正常

### 数据获取失败
- 检查网络连接
- 检查东方财富API是否可访问
- 等待一段时间后重试

### 邮件发送失败
- 检查邮件配置是否正确
- 检查发送邮箱的SMTP服务是否开启
- 检查网络连接

## 技术支持

如果遇到问题，请检查系统日志输出，或联系技术支持。
