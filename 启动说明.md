# 微盘股指数分析系统 - 启动说明

## 🚀 快速启动（推荐）

### Windows 系统

**方法一：使用 Python 启动脚本（最简单、最可靠）**

1. 双击运行 `启动系统.py`
   - 或者在命令行运行：`python 启动系统.py`
2. 等待系统自动启动后端和前端服务
3. 浏览器会自动打开系统页面

**方法二：使用一键启动批处理文件**

1. 双击运行 `一键启动.bat`
2. 等待系统自动启动后端和前端服务
3. 浏览器会自动打开系统页面

**方法三：使用 Python 自动启动脚本（高级）**

```bash
python auto_start.py
```

### 手动启动

如果自动启动脚本无法使用，可以手动启动：

#### 1. 启动后端服务

```bash
# 进入后端目录
cd web\backend

# 启动服务
python start_backend.py
```

#### 2. 启动前端服务

```bash
# 进入前端目录
cd web\frontend

# 启动 HTTP 服务器
python -m http.server 5500
```

#### 3. 打开浏览器

访问：http://localhost:5500/web/frontend/index.html

## 📋 详细说明

### 系统架构

- **后端服务**：Flask 框架，运行在 `http://localhost:5000`
- **前端服务**：React + Tailwind CSS，运行在 `http://localhost:5500`
- **数据更新**：每 30 分钟自动刷新

### 端口说明

- **5000 端口**：后端 API 服务
- **5500 端口**：前端静态资源服务

### 常见问题

#### Q1: 端口被占用怎么办？

如果 5000 或 5500 端口被占用，可以：

1. 关闭占用端口的程序
2. 或者修改启动脚本中的端口号

#### Q2: 后端服务启动失败？

检查以下事项：

1. Python 3.8+ 已安装
2. 安装依赖：`pip install flask flask-cors`
3. 查看错误日志

#### Q3: 前端页面无法访问？

1. 确认前端服务已启动（端口 5500）
2. 检查防火墙设置
3. 清除浏览器缓存后重试
