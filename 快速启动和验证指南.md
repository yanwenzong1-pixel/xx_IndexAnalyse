## 常见问题

### Q1: API 返回 "service_not_initialized"
**原因**：历史回溯服务初始化失败

**解决方案**：
1. 检查后端日志，查看初始化错误信息
2. 确保数据获取成功（`analyzer.fetch_data()`）
3. 重启服务

### Q2: 前端显示"数据加载失败"
**原因**：无法连接到后端 API

**解决方案**：
1. 确认后端服务已启动
2. 检查 API 地址是否正确（`http://localhost:5000`）
3. 检查 CORS 配置（如使用 flask-cors）

### Q3: 计算时间过长（> 1s）
**原因**：数据量大或计算资源不足

**解决方案**：
1. 减少回溯天数（`days` 参数）
2. 优化计算逻辑（增量计算）
3. 引入后端缓存（Redis）

### Q4: 数据范围为 0-100%，而非 20-80%
**原因**：前端数据转换错误

**解决方案**：
1. 检查数据转换代码：`t1Prob: parseFloat((item.prob_t1 * 100).toFixed(2))`
2. 确保后端返回的是小数（0-1），而非百分比（0-100）
