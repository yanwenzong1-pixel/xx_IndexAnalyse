### 方式二：一键部署脚本

```bash

```

#### Linux/Mac系统

```bash
# 创建并运行部署脚本
cat > deploy.sh << 'EOF'
#!/bin/bash
echo "========================================"
echo "微盘股指数分析系统 - 部署"
echo "========================================"
pip3 install flask flask-cors requests pandas numpy scipy
cd web/backend
python3 simple_app.py
EOF

chmod +x deploy.sh
./deploy.sh
```

## 🔧 故障排查

### 问题 1: flask-cors 未安装

**错误**: `ModuleNotFoundError: No module named 'flask_cors'`

**解决**:
```bash
pip install flask-cors
```

### 问题 2: 端口被占用

**错误**: `OSError: [WinError 10048] 端口被占用`

**解决**:
```bash
# 修改端口（在 simple_app.py 中）
app.run(host='0.0.0.0', port=5001)  # 改为 5001
```

### 问题 3: API 数据获取失败

**错误**: `数据获取失败：...`

**解决**:
- 检查网络连接
- 确认可以访问东方财富 API
- 检查防火墙设置

### 问题 4: 依赖安装失败

**错误**: `Could not find a version that satisfies the requirement`

**解决**:
```bash
# 升级 pip
python -m pip install --upgrade pip

# 使用国内镜像
pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple
```
