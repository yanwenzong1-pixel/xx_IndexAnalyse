## 六、问题排查

### 6.1 常见问题

**问题 1**: 模块导入失败
```
ModuleNotFoundError: No module named 'pandas'
```
**解决**: 
```bash
pip install pandas numpy scipy requests flask
```

**问题 2**: 端口被占用
```
OSError: [WinError 10048] 通常每个套接字地址 (协议/网络地址/端口) 只允许使用一次
```
**解决**: 
```bash
# 修改端口
cd web/backend
python app.py --port 5001
```

**问题 3**: API 数据获取失败
```
数据获取失败：...
```
**解决**: 
- 检查网络连接
- 确认东方财富 API 可访问
- 检查防火墙设置

**问题 4**: 缓存目录权限问题
```
PermissionError: [WinError 5] 拒绝访问
```
**解决**: 
- 以管理员身份运行
- 或修改缓存目录为用户目录
