## 快速验证脚本

创建文件 `start_backend.bat`：

```batch
@echo off
cd /d "%~dp0"
echo 正在启动后端服务...
python web/backend/simple_app.py
pause
```

双击运行此文件即可启动后端服务。

## 常见问题

### Q1: 端口 5000 被占用？
**错误信息**：`Address already in use`

**解决方案**：
```bash
# 方法 1：终止占用端口的进程
netstat -ano | findstr :5000
taskkill /PID <进程 ID> /F

# 方法 2：修改端口号
# 编辑 simple_app.py，将 app.run(port=5000) 改为 app.run(port=5001)
```

### Q2: Python 环境问题？
**错误信息**：`ModuleNotFoundError`

**解决方案**：
```bash
# 安装依赖
pip install flask flask-cors pandas numpy
```

### Q3: 数据文件缺失？
**错误信息**：`File not found`

**解决方案**：
确保数据文件存在于正确的位置
